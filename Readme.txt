---------------------------------
Relax Now - SIT313 Assignment 1
---------------------------------

Relax now is a mindfulness and meditation application used to help users in their meditation practice
by allowing them to layer on sounds of their choosing, and record how they are feeling before the meditation.

---------------------------------
Prerequisites
---------------------------------
To run my application you need to open it in Visual Studio 2017. 
To allow for the record feature to work you must go into android preferences and check "Allow Recording", and "Allow Saving to Device".
---------------------------------
Installing and Running
---------------------------------
Simply clone the github file
Open file in Visual Studios
Allow proper permissions for device or emulator
Build to emulator or device.
